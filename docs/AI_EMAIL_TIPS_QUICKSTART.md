## 步骤 1: 确保 AI 已配置

如果你还没有配置 AI，请先完成以下步骤：

### 推荐方案：OpenAI（国际用户）

1. 访问 https://platform.openai.com/api-keys
2. 获取 API Key（格式：`sk-proj-xxx`）
3. 在 HomeBug 中：**设置 → 系统配置 → AI 配置**
4. 填写配置：
   ```
   是否启用 AI 功能: ✅ 开启
   AI 提供商: openai
   AI 模型名称: gpt-4o-mini
   AI API Key: sk-proj-xxx
   AI 温度参数: 0.7  ⚠️ 重要：调高温度让文案更有创意
   ```
5. 保存配置

### 推荐方案：豆包（国内用户）

1. 访问 https://console.volcengine.com/ark
2. 创建推理接入点，获取 API Key
3. 在 HomeBug 中配置：
   ```
   是否启用 AI 功能: ✅ 开启
   AI 提供商: doubao
   AI 模型名称: doubao-pro-32k
   AI API Key: (你的 API Key)
   AI API Base URL: https://ark.cn-beijing.volces.com/api/v3
   AI 温度参数: 0.7
   ```

💡 **提示**：温度参数越高（0.7-1.0），文案越有创意和幽默感！

## 步骤 2: 测试邮件功能

### 方法 1: 手动触发（推荐）

1. 以管理员身份登录
2. 访问 `/api/cron`（GET 请求查看状态）
3. 发送 POST 请求到 `/api/cron` 触发任务
4. 检查邮箱，查看是否收到带 AI 文案的邮件

### 方法 2: 等待定时任务

系统会根据配置的 Cron 表达式自动发送邮件（默认每天凌晨 1 点）。

## 常见问题

### Q: 邮件中没有 AI 文案？

**A**: 检查以下几点：

1. AI 功能是否已启用？
2. API Key 是否有效？
3. 账户余额是否充足？
4. 查看控制台是否有错误日志

### Q: AI 文案不够幽默？

**A**:

1. 调高 AI 温度参数（0.7-1.0）
2. 更换模型（如 gpt-4o）
3. 等待几次生成，AI 会有不同的创意

### Q: 成本会很高吗？

**A**: 非常低！

- 单次邮件（20 个物品）：约 $0.0004（0.3 分）
- 每月（每天一次）：约 $0.01（不到 1 毛钱）

### Q: AI 不可用会影响邮件发送吗？

**A**: 不会！

- AI 失败时自动降级到默认文案
- 邮件正常发送
- 不影响核心功能

## 自定义文案风格

如果你想调整文案风格，可以修改 `src/app/api/cron/route.ts` 中的 `generateAiTipsForItems()` 函数。

### 示例：更正式的风格

找到提示词部分，修改为：

```typescript
const prompt = `你是一个专业的家庭管家助手。请为以下物品生成简洁、专业的提示文案。

要求：
1. 每个物品一句话，10-20字
2. 语气正式、专业
3. 直接说明问题和建议
4. 避免过度幽默和网络用语
...
```

### 示例：更温馨的风格

```typescript
const prompt = `你是一个温柔体贴的家庭管家助手。请为以下物品生成温馨、贴心的提示文案。

要求：
1. 每个物品一句话，15-25字
2. 语气温和、关怀
3. 像家人提醒一样
4. 避免调侃和夸张
...
```
