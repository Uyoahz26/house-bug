## 📋 使用方法

### 快速开始（以阿里通义千问为例）

1. **注册并获取 API Key**
   - 访问：https://dashscope.aliyun.com/
   - 注册账号并实名认证
   - 创建 API Key：https://dashscope.console.aliyun.com/apiKey
   - 充值 ¥10-50（支付宝）

2. **在系统中配置**
   - 进入：设置 → 系统配置 → AI 配置
   - 填写以下信息：
     ```
     AI 启用: 开启
     AI 提供商: custom
     AI 模型名称: qwen-vl-plus
     AI API Key: sk-xxxxxxxxxxxxxxxx
     AI API Base URL: https://dashscope.aliyuncs.com/compatible-mode/v1
     ```
   - 保存配置

3. **测试**
   - 进入物品管理页面
   - 上传商品图片
   - 点击识别按钮
   - 查看识别结果

## 🔧 技术实现

### 当前架构

```
用户上传图片
    ↓
前端发送到 /api/items/ocr
    ↓
后端读取 AI 配置（ai.provider = custom）
    ↓
使用 OpenAiAdapter（兼容 OpenAI API 格式）
    ↓
调用国内厂商 API（通过 ai.api_base）
    ↓
返回识别结果
```

### 配置示例

```typescript
// 数据库配置
ai.enabled = "1";
ai.provider = "custom";
ai.model = "qwen-vl-plus";
ai.api_key = "sk-xxxxxxxxxxxxxxxx";
ai.api_base = "https://dashscope.aliyuncs.com/compatible-mode/v1";
ai.temperature = "0.1";
ai.max_tokens = "2000";
ai.timeout = "30000";
```
