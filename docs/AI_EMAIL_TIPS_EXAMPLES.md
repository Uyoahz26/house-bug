## 文案风格分析

### 幽默手法

1. **夸张法**
   - "这瓶洗发水已经修炼成精了"
   - "牛奶变酸奶，酸奶变奶酪"
   - "再不囤货，就要用树叶了"

2. **谐音梗**
   - "雪饼不旺了"
   - "蚝油变蚝臭"
   - "香皂不香了"

3. **网络流行语**
   - "已经凉透了🥶"
   - "外卖走起？"
   - "这可是刚需啊！"

4. **拟人化**
   - "松鼠都等急了"
   - "鸭脖在召唤你"
   - "老干妈都老到不能再老了"

5. **场景化**
   - "今晚来个酸奶派对吧！"
   - "今晚宵夜就它了！"
   - "早餐没奶喝，只能干啃面包了"

### 情感色彩

- **已过期**：调侃 + 提醒丢弃
- **即将过期**：鼓励 + 行动建议
- **库存不足**：紧迫 + 补货提醒

## 自定义文案风格

如果你想调整文案风格，可以修改 `generateAiTipsForItems()` 函数中的提示词：

### 更正式的风格

```typescript
const prompt = `请为以下物品生成简洁、专业的提示文案。
要求：
1. 语气正式、专业
2. 直接说明问题和建议
3. 避免过度幽默
...
```

### 更温馨的风格

```typescript
const prompt = `请为以下物品生成温馨、贴心的提示文案。
要求：
1. 语气温和、关怀
2. 像家人提醒一样
3. 避免调侃和夸张
...
```

### 更简洁的风格

```typescript
const prompt = `请为以下物品生成简短、直接的提示文案。
要求：
1. 每条不超过 15 字
2. 直接说明问题
3. 避免修饰和解释
...
```

## 总结

AI 生成的文案让库存提醒变得更加生动有趣，提升了用户体验。通过合理的提示词设计，可以生成各种风格的文案，满足不同用户的需求。
