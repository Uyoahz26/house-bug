# HomeBug OpenClaw - AI 幽默回复功能

## 🎭 功能概述

HomeBug OpenClaw Skill 现在支持 **AI 实时生成幽默回复**！每次操作都会得到独特、有趣的回复消息，让家庭物资管理变得更加生动有趣。

---

## ✨ 特色

### 1. 方大同式莫名其妙 🎵

回复风格像方大同的歌词一样，看似莫名其妙但又很有意境：

```
"纸巾在你手中化作云烟，还剩 5 包在等待命运的召唤 ✨"
"冰箱打开是惊喜，关上是回忆～里面藏着 5 个小秘密 🧊"
"时光不饶人，这些东西已经在岁月中迷失了方向 😢"
```

### 2. 网络热梗融入 🔥

适当使用流行的网络用语，让回复更接地气：

- **yyds**（永远的神）
- **绝绝子**（太好了）
- **躺平**（佛系状态）
- **家人们**（亲切称呼）
- **这波属于是**（网络流行句式）
- **富得流油**（形容很多）

示例：

```
"鸡蛋大军集结完毕！现在有 15 个蛋蛋在冰箱里躺平，这波属于是囤货 yyds 了 🥚"
"纸巾补给到位！现在有 8 包在卫生间待命，富得流油啊家人们 📦"
```

### 3. 智能情境感知 🧠

AI 会根据不同情况调整回复风格：

**数量多** → 调侃"囤货小能手"

```
"厨房里的囤货小能手上线！6 件物品整整齐齐，这波属于是居家达人了 🍳"
```

**数量少** → 说"极简主义"

```
"卫生间走的是极简主义路线，4 件物品刚刚好，不多不少 🚿"
```

**数量为 0** → 幽默提醒

```
"纸巾库存清零！这波属于是空空如也了，建议火速补货 🚨"
```

**有过期物品** → 诗意调侃

```
"时光不饶人，这些东西已经在岁月中迷失了方向 😢"
```

**无过期物品** → 积极鼓励

```
"恭喜！你的冰箱比你的人生还要新鲜，一件过期的都没有 ✨"
```

## 🔧 技术实现

### AI Prompt 设计

每个操作类型都有专门的 AI Prompt：

#### 1. 减少/增加物品

```typescript
const operationPrompt = `生成一条幽默有趣的消息，告诉用户已将"${itemName}"的数量${operationType} ${quantity} ${unit}，当前剩余 ${newQuantity} ${unit}。要求：
1. 语言风格像方大同的歌词那样莫名其妙但又很有意境
2. 可以适当使用网络热梗
3. 如果数量变为0，要调侃一下"空了"的状态
4. 如果数量很多，可以说"富得流油"之类的
5. 控制在80字以内
6. 只返回消息文本，不要其他内容`;
```

#### 2. 查询已过期物品

```typescript
const expiredPrompt = `生成一条幽默有趣的消息，告诉用户发现了 ${count} 件过期物品。要求：
1. 语言风格像方大同的歌词那样莫名其妙但又很有意境
2. 可以适当使用网络热梗
3. 要有一点点调侃但不要太过分
4. 开头部分控制在80字以内
5. 只返回开头消息文本，不要包含物品列表`;
```

#### 3. 查询即将过期物品

```typescript
const expiringPrompt = `生成一条幽默有趣的消息，告诉用户发现了 ${count} 件物品将在 ${days} 天内过期。要求：
1. 语言风格像方大同的歌词那样莫名其妙但又很有意境
2. 可以适当使用网络热梗
3. 要有紧迫感但不要太焦虑，可以调侃一下
4. 开头部分控制在80字以内
5. 只返回开头消息文本，不要包含物品列表`;
```

#### 4. 查询特定位置物品

```typescript
const locationPrompt = `生成一条幽默有趣的消息，告诉用户"${location}"里有 ${count} 件物品。要求：
1. 语言风格像方大同的歌词那样莫名其妙但又很有意境
2. 可以适当使用网络热梗
3. 可以根据物品数量调侃一下（多了就说"囤货小能手"，少了就说"极简主义"）
4. 开头部分控制在80字以内
5. 只返回开头消息文本，不要包含物品列表`;
```

### 代码示例

```typescript
// 使用 AI 生成幽默的操作结果消息
const operationMessages: AiMessage[] = [
  { role: "user", content: operationPrompt },
];

const operationResponse = await adapter.chat(operationMessages, aiConfig);
const message = operationResponse.content.trim();

return NextResponse.json({
  success: true,
  message,
  operation: {
    type: parsedAction.type,
    itemName: targetItem.name,
    itemId: targetItem.id,
    previousQuantity,
    newQuantity,
    unit: targetItem.unit,
  },
});
```

## 🎨 回复风格指南

### 语言特点

1. **诗意化表达**
   - ❌ "纸巾用完了"
   - ✅ "纸巾在你手中化作云烟"

2. **意境优先**
   - ❌ "冰箱里有 5 件物品"
   - ✅ "冰箱打开是惊喜，关上是回忆～里面藏着 5 个小秘密"

3. **适度热梗**
   - ❌ "鸡蛋增加了 10 个"
   - ✅ "鸡蛋大军集结完毕！这波属于是囤货 yyds 了"

4. **情境感知**
   - 数量多 → "富得流油"、"囤货小能手"
   - 数量少 → "极简主义"、"刚刚好"
   - 数量为 0 → "空空如也"、"火速补货"

### 情绪把控

- ✅ **幽默但不过分**：调侃但保持温度
- ✅ **积极正能量**：即使调侃也是积极的
- ✅ **有趣不油腻**：网络热梗适度使用
- ✅ **诗意不做作**：莫名其妙但有意境

### 1. 每次都有惊喜

AI 实时生成，每次回复都不同：

```

```

查询空位置时的特殊回复：

