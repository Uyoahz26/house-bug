## 基本操作

### 1. 减少物品数量（消耗）

适用于日常使用、消耗物品的场景。

```
你：我用了一包纸
OpenClaw：已将"纸"的数量减少 1 包，当前剩余 5 包

你：吃了 2 个苹果
OpenClaw：已将"苹果"的数量减少 2 个，当前剩余 8 个

你：喝了一瓶牛奶
OpenClaw：已将"牛奶"的数量减少 1 瓶，当前剩余 2 瓶

你：用掉了 3 个鸡蛋
OpenClaw：已将"鸡蛋"的数量减少 3 个，当前剩余 12 个
```

### 2. 增加物品数量（补充）

适用于购买、补充物品的场景。

```
你：买了 10 个鸡蛋
OpenClaw：已将"鸡蛋"的数量增加 10 个，当前剩余 15 个

你：补充了 5 包纸巾
OpenClaw：已将"纸巾"的数量增加 5 包，当前剩余 8 包

你：购买了 2 瓶酱油
OpenClaw：已将"酱油"的数量增加 2 瓶，当前剩余 3 瓶

你：添加了 1 桶食用油
OpenClaw：已将"食用油"的数量增加 1 桶，当前剩余 2 桶
```

### 3. 查询特定物品

查询家中某个物品的库存情况。

```
你：家里还有多少牛奶
OpenClaw：您家目前有 3 瓶牛奶（1L装），存放在冰箱

你：剩余多少鸡蛋
OpenClaw：您家目前有 15 个鸡蛋，存放在冰箱

你：查询纸巾库存
OpenClaw：您家目前有 8 包纸巾（维达），存放在卫生间

你：还有多少大米
OpenClaw：您家目前有 10 kg 大米（五常大米），存放在厨房
```

## 高级查询

### 4. 查询已过期物品

检查家中有哪些物品已经过期。

```
你：有哪些东西已经过期了
OpenClaw：发现 2 件已过期物品：
- 牛奶（蒙牛）：1 瓶，过期日期 2026-04-20，存放在冰箱
- 面包（桃李）：1 袋，过期日期 2026-04-22，存放在厨房

你：过期的物品有哪些
OpenClaw：太好了！目前没有已过期的物品。

你：哪些东西过期了
OpenClaw：发现 1 件已过期物品：
- 酸奶（伊利）：2 盒，过期日期 2026-04-18，存放在冰箱
```

### 5. 查询即将过期物品

查看哪些物品快要过期，提前规划使用。

```

OpenClaw：发现 2 件物品将在 7 天内过期：

- 面包（桃李）：1 袋，还有 4 天过期，存放在厨房

你：即将过期的物品

你：马上要过期的东西
```

## 实用技巧

### 技巧 1：模糊匹配

OpenClaw 使用 AI 智能匹配，支持模糊查询：

```
你：用了一包纸
→ 可以匹配"纸巾"、"卫生纸"、"面巾纸"等

你：买了牛奶
→ 可以匹配"纯牛奶"、"鲜牛奶"、"酸奶"等

你：还有多少油
→ 可以匹配"食用油"、"花生油"、"橄榄油"等
```

### 技巧 2：自然语言表达

使用自然的口语化表达：

```
✅ "我用了一包纸"
✅ "吃了 2 个苹果"
✅ "买了 10 个鸡蛋"
✅ "家里还有多少牛奶"
✅ "冰箱里有什么"
✅ "哪些东西快过期了"
```

### 技巧 3：定期检查

建议定期使用以下查询：

**每周检查**：

```
你：哪些东西快过期了
你：有哪些东西已经过期了
```

**购物前检查**：

```
你：冰箱里有什么
你：厨房有哪些东西
你：还有多少XX（具体物品）
```

**整理前检查**：

```
你：有哪些东西已经过期了
你：储物间里面有什么东西
```

### 技巧 4：位置管理

合理使用位置查询功能：

```
常用位置：
- 冰箱：生鲜、乳制品、饮料
- 厨房：调料、粮油、干货
- 卫生间：洗护用品、纸巾
- 储物间：囤货、清洁用品
- 客厅：零食、饮料
- 卧室：个人用品
```

### 技巧 5：过期提醒

利用过期查询功能：

```
每天早上：
你：有哪些东西已经过期了

每周一次：
你：哪些东西快过期了

购物前：
你：哪些东西快过期了
→ 优先消耗快过期的物品，避免浪费
```

## 常见问题

### Q1：找不到物品怎么办？

**A**：确保物品已在 HomeBug 中添加。如果已添加但找不到，尝试使用更精确的名称。

```
❌ "纸"（太模糊）
✅ "纸巾" 或 "卫生纸"（更精确）
```

### Q2：如何查看所有物品？

**A**：目前 OpenClaw 不支持查看所有物品，建议登录 HomeBug 网页端查看完整列表。

### Q3：可以一次操作多个物品吗？

**A**：目前不支持，需要分别操作：

```
你：我用了一包纸
OpenClaw：已将"纸"的数量减少 1 包，当前剩余 5 包

你：还用了 2 个鸡蛋
OpenClaw：已将"鸡蛋"的数量减少 2 个，当前剩余 13 个
```

### Q4：如何修改物品信息？

**A**：OpenClaw 目前只支持数量操作，修改其他信息（如名称、位置、过期日期）需要登录 HomeBug 网页端。
