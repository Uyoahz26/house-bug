# HomeBug OpenClaw 功能增强总结

## 📅 更新日期：2026-04-24

---

## 🎯 增强概述

本次更新为 HomeBug OpenClaw Skill 添加了三个重要的高级查询功能，使其从基础的库存增减工具升级为全功能的智能家庭物资管理助手。

---

## ✨ 新增功能

### 1. 查询已过期物品 ⚠️

**功能描述**：

- 一键查询所有已过期的物品
- 显示过期日期和存放位置
- 帮助用户及时清理过期物资

**触发词**：

- "有哪些东西已经过期了"
- "过期的物品有哪些"
- "哪些东西过期了"

**使用示例**：

```
你：有哪些东西已经过期了
OpenClaw：发现 2 件已过期物品：
- 牛奶（蒙牛）：1 瓶，过期日期 2026-04-20，存放在冰箱
- 面包（桃李）：1 袋，过期日期 2026-04-22，存放在厨房
```

**应用场景**：

- 每周健康检查
- 大扫除前清理
- 整理冰箱前检查

---

### 2. 查询即将过期物品 ⏰

**功能描述**：

- 查询未来 7 天内即将过期的物品
- 显示剩余天数和存放位置
- 按过期日期排序（最快过期的在前）
- 帮助用户优先消耗快过期物品

**触发词**：

- "哪些东西快过期了"
- "快到期的有什么"
- "即将过期的物品"
- "马上要过期的东西"

**使用示例**：

```
你：哪些东西快过期了
OpenClaw：发现 3 件物品将在 7 天内过期：
- 酸奶（伊利）：2 盒，还有 3 天过期，存放在冰箱
- 鸡蛋：8 个，还有 5 天过期，存放在冰箱
- 火腿肠：1 包，还有 6 天过期，存放在厨房
```

**应用场景**：

- 购物前检查（避免重复购买快过期物品）
- 制定饮食计划（优先消耗快过期食材）
- 每周定期检查

---

### 3. 查询特定位置物品 📍

**功能描述**：

- 查询特定位置的所有物品
- 自动显示过期信息（已过期或即将过期）
- 支持模糊匹配位置名称
- 帮助用户了解各个位置的物资情况

**触发词**：

- "冰箱里有什么"
- "厨房有哪些东西"
- "卫生间有什么"
- "XX位置的物品"

**使用示例**：

```
你：冰箱里有什么
OpenClaw：冰箱里有 5 件物品：
- 牛奶（蒙牛）：2 瓶
- 酸奶（伊利）：2 盒，还有 3 天过期
- 鸡蛋：8 个，还有 5 天过期
- 黄油：1 盒
- 啤酒：6 瓶
```

**应用场景**：

- 做饭前检查食材
- 购物前查看库存
- 整理收纳前盘点

## 🔧 技术实现

### API 增强

**新增操作类型**：

```typescript
type OperationType =
  | "decrease" // 减少数量
  | "increase" // 增加数量
  | "query" // 查询特定物品
  | "query_expired" // 查询已过期物品 ⭐ 新增
  | "query_expiring" // 查询即将过期物品 ⭐ 新增
  | "query_location"; // 查询特定位置物品 ⭐ 新增
```

**新增参数**：

```typescript
interface ParsedAction {
  type: OperationType;
  itemName?: string; // 物品名称（可选）
  quantity: number; // 数量
  location?: string; // 位置名称 ⭐ 新增
  daysThreshold?: number; // 天数阈值 ⭐ 新增（默认7天）
}
```

### AI 提示词优化

增强了 AI 解析能力，支持更多自然语言表达：

```typescript
const systemPrompt = `你是一个智能物资管理助手。解析用户的自然语言指令...

操作类型：
- decrease: 减少/消耗/用了/吃了/喝了
- increase: 增加/买了/补充了/添加了
- query: 查询/还有多少/剩余（查询特定物品）
- query_expired: 查询已过期的物品 ⭐
- query_expiring: 查询即将过期的物品（快过期了/快到期了）⭐
- query_location: 查询特定位置的物品（冰箱里有什么/厨房有哪些）⭐
`;
```

### 数据处理逻辑

#### 1. 查询已过期物品

```typescript
const expiredItems = await listItems(db, {
  status: "expired",
  limit: 1000,
  offset: 0,
});
```

#### 2. 查询即将过期物品

```typescript
const expiringItems = allActiveItems
  .filter((item) => {
    if (!item.expiry_date) return false;
    const expiryDate = new Date(item.expiry_date);
    return expiryDate >= today && expiryDate <= thresholdDate;
  })
  .sort(
    (a, b) =>
      new Date(a.expiryDate!).getTime() - new Date(b.expiryDate!).getTime(),
  );
```

#### 3. 查询特定位置物品

```typescript
const locationItems = allItems.filter(
  (item) =>
    item.location &&
    item.location.toLowerCase().includes(location.toLowerCase()),
);
```

## 🎯 使用场景

### 场景 1：每日健康检查 🏥

**目标**：确保家中没有过期物品

```
早上起床后：
你：有哪些东西已经过期了
OpenClaw：太好了！目前没有已过期的物品。
```

### 场景 2：每周定期检查 📅

**目标**：提前规划，避免浪费

```
每周日晚上：
你：哪些东西快过期了
OpenClaw：发现 3 件物品将在 7 天内过期：
- 酸奶（伊利）：2 盒，还有 3 天过期，存放在冰箱
- 鸡蛋：8 个，还有 5 天过期，存放在冰箱
- 火腿肠：1 包，还有 6 天过期，存放在厨房

→ 制定下周饮食计划，优先消耗这些食材
```

### 场景 3：购物前准备 🛒

**目标**：避免重复购买，优先消耗库存

```
准备去超市前：
你：冰箱里有什么
OpenClaw：冰箱里有 5 件物品：
- 牛奶（蒙牛）：2 瓶
- 酸奶（伊利）：2 盒，还有 3 天过期
- 鸡蛋：8 个，还有 5 天过期
- 黄油：1 盒
- 啤酒：6 瓶

你：哪些东西快过期了
OpenClaw：发现 3 件物品将在 7 天内过期...

→ 决定不买牛奶和鸡蛋，优先消耗快过期的酸奶
```

### 场景 4：做饭前检查 🍳

**目标**：了解可用食材

```
准备做晚饭：
你：冰箱里有什么
OpenClaw：冰箱里有 5 件物品...

你：厨房有哪些调料
OpenClaw：厨房里有 4 件物品：
- 酱油（海天）：1 瓶
- 食用油（金龙鱼）：1 桶
- 盐：1 袋
- 糖：1 袋

→ 根据现有食材和调料决定做什么菜
```

### 场景 5：大扫除前清理 🧹

**目标**：清理过期物品，整理收纳

```
周末大扫除：
你：有哪些东西已经过期了
OpenClaw：发现 2 件已过期物品...

你：储物间里面有什么东西
OpenClaw：储物间里有 3 件物品...

你：卫生间有什么
OpenClaw：卫生间里有 4 件物品...

→ 清理过期物品，整理各个位置的物资
```
